## ­ЪЊЮ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃарЃБрЃџрЃљрЃЊ рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў FullStack рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЌрЃћрЃЏрЃљрЃф рЃЏрЃЮрЃњрЃќрЃљрЃБрЃарЃЮрЃЉрЃљрЃљ.  
рЃцрЃарЃЮрЃюрЃбрЃћрЃюрЃЊрЃерЃў рЃњрЃЋрЃ«рЃЋрЃЊрЃћрЃЉрЃљ:

- рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў
- рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃў
- рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃў

### ­ЪДГ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў

- рЃърЃўрЃарЃЋрЃћрЃџ рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў РђЊ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў & Navbar
- рЃЏрЃћрЃЮрЃарЃћ рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў РђЊ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃћрЃџрЃќрЃћрЃф рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ рЃўрЃдрЃћрЃЉрЃЌ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА
- рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў РђЊ рЃърЃљрЃбрЃљрЃарЃљ рЃЌрЃљрЃЏрЃљрЃерЃў: рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ рЃњрЃўрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃЌ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃљрЃЊрЃњрЃўрЃџрЃА рЃцрЃЮрЃбрЃЮрЃЌрЃў рЃЊрЃљ рЃљрЃдрЃгрЃћрЃарЃўрЃЌ
- рЃЏрЃћрЃЮрЃЌрЃ«рЃћ рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў РђЊ рЃАрЃџрЃљрЃўрЃЊрЃћрЃарЃў рЃцрЃЮрЃбрЃЮрЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃЌ
- рЃЉрЃЮрЃџрЃЮ рЃюрЃљрЃгрЃўрЃџрЃерЃў РђЊ Footer

### ­Ъћљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ & рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ

- Navbar-рЃўрЃА **Human Icon-рЃќрЃћ** рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ рЃЕрЃюрЃЊрЃћрЃЉрЃљ DropDown рЃЏрЃћрЃюрЃўрЃБ, рЃАрЃљрЃўрЃЊрЃљрЃюрЃљрЃф рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃљрЃўрЃарЃЕрЃўрЃЮрЃЌ Sign Up/Login
- Book рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃўрЃАрЃљрЃА рЃњрЃљрЃЊрЃљрЃЊрЃўрЃ«рЃљрЃарЃЌ Login рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
- рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЊрЃљрЃ«рЃЋрЃљрЃџрЃЌ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ, рЃЊрЃљ book-рЃќрЃћ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃЊрЃљрЃГрЃћрЃарЃўрЃАрЃљрЃА рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃЊрЃљрЃ«рЃЋрЃљрЃџрЃЌ рЃбрЃБрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА/рЃЊрЃљрЃ»рЃљрЃЋрЃерЃюрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ

### ­ЪД│ Booking рЃњрЃЋрЃћрЃарЃЊрЃў

- рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃбрЃБрЃарЃў
- рЃўрЃ«рЃўрЃџрЃЮрЃА рЃАрЃ«рЃЋрЃљ рЃбрЃБрЃарЃћрЃЉрЃў: `Other Tours`
- рЃюрЃљрЃ«рЃЮрЃА рЃарЃЮрЃЏрЃћрЃџ рЃбрЃБрЃарЃћрЃЉрЃќрЃћрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў: `Joined Tours`
- рЃўрЃ«рЃўрЃџрЃЮрЃА рЃбрЃБрЃарЃќрЃћ рЃЏрЃДрЃЮрЃцрЃў рЃАрЃ«рЃЋрЃљ рЃгрЃћрЃЋрЃарЃћрЃЉрЃў
