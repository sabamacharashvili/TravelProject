# ­ЪїЇ TravelProject

**TravelProject** Рђћ рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃарЃБрЃџрЃцрЃљрЃАрЃЮрЃЋрЃљрЃюрЃў FullStack responsive рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃЏрЃЮрЃњрЃќрЃљрЃБрЃарЃЮрЃЉрЃўрЃА рЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃљрЃА. рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф Frontend-, рЃўрЃАрЃћ Backend-рЃюрЃљрЃгрЃўрЃџрЃўрЃЌ рЃЊрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА рЃерЃћрЃљрЃарЃЕрЃўрЃЮрЃА, рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃљрЃю рЃЊрЃљрЃћрЃЏрЃљрЃбрЃЮрЃА рЃбрЃБрЃарЃћрЃЉрЃќрЃћ.

---

## ­ЪЊї рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

### ­Ъїљ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў (Home Page)

- Рюћ№ИЈ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃАрЃћрЃЦрЃфрЃўрЃљ рЃцрЃЮрЃюрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃўрЃЌрЃљ рЃЊрЃљ Navbar-рЃўрЃЌ
- Рюћ№ИЈ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃБрЃарЃўрЃАрЃбрЃБрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃцрЃЮрЃбрЃЮрЃњрЃљрЃџрЃћрЃарЃћрЃљ, рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃўрЃЌ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃўрЃАрЃљрЃА
- Рюћ№ИЈ рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃўрЃАрЃбрЃБрЃарЃў рЃЌрЃљрЃЏрЃљрЃерЃў "рЃарЃЮрЃЏрЃћрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃў рЃљрЃЋрЃўрЃарЃЕрЃўрЃЮ?" Рђћ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ
- Рюћ№ИЈ рЃЉрЃЮрЃџрЃЮ рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў рЃАрЃџрЃљрЃўрЃЊрЃћрЃа-рЃбрЃўрЃърЃўрЃА рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃўрЃА рЃЕрЃљрЃарЃЕрЃЮ
- Рюћ№ИЈ рЃАрЃарЃБрЃџрЃљрЃЊ рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў Рђћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃарЃЮрЃњрЃЮрЃарЃф Desktop-, рЃўрЃАрЃћ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃа рЃћрЃЎрЃарЃљрЃюрЃћрЃЉрЃќрЃћ
- Рюћ№ИЈ Footer-рЃў рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃџрЃўрЃюрЃЎрЃўрЃЌрЃљ рЃЊрЃљ рЃАрЃЮрЃфрЃўрЃљрЃџрЃБрЃа рЃЦрЃАрЃћрЃџрЃћрЃЉрЃЌрЃљрЃю рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃЌ

---

### ­Ъћљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ

- ­ЪћЇ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃАрЃћрЃЦрЃфрЃўрЃўрЃА Navbar-рЃерЃў рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ Human Icon
- ­ЪЉц рЃЏрЃљрЃАрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ рЃЕрЃюрЃЊрЃћрЃЉрЃљ DropDown рЃЏрЃћрЃюрЃўрЃБ, рЃАрЃљрЃЊрЃљрЃф рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ Login/Sign Up-рЃА рЃљрЃарЃЕрЃћрЃЋрЃљ
- ­ЪЊџ рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ Book рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ рЃџрЃЮрЃњрЃўрЃю рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
- РюЁ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃљрЃю рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА Home рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃЊрЃљ рЃўрЃЦрЃўрЃЊрЃљрЃю рЃбрЃБрЃарЃўрЃА рЃЊрЃљрЃ»рЃљрЃЋрЃерЃюрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ

---

### ­ЪЊЁ рЃбрЃБрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃЊрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЉрЃљ

- РъЋ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃЦрЃЏрЃюрЃўрЃА рЃбрЃБрЃарЃА Рђћ рЃљрЃЊрЃњрЃўрЃџрЃўрЃА, рЃљрЃдрЃгрЃћрЃарЃўрЃАрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃўрЃЌ
- ­ЪЉЦ рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЊрЃљрЃћрЃЏрЃљрЃбрЃЮрЃА рЃАрЃ«рЃЋрЃљ рЃбрЃБрЃарЃћрЃЉрЃќрЃћ
- ­ЪћЇ рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃўрЃ«рЃўрЃџрЃЮрЃА:
  - Other Tours Рђћ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃћрЃа рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃбрЃБрЃарЃћрЃЉрЃў
  - Joined Tours Рђћ рЃбрЃБрЃарЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃќрЃћрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ

## РџЎ№ИЈ Backend (рЃАрЃћрЃарЃЋрЃћрЃарЃБрЃџрЃў рЃюрЃљрЃгрЃўрЃџрЃў)

рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў:

- ­ЪЪб **Node.js** РђЊ рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА
- РџЎ№ИЈ **Express.js** РђЊ RESTful API-рЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ
- ­ЪЇЃ **MongoDB** РђЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў, рЃбрЃБрЃарЃћрЃЉрЃў рЃЊрЃљ рЃљ.рЃе.)
- ­Ъћњ рЃЊрЃљрЃфрЃБрЃџрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ/рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ
